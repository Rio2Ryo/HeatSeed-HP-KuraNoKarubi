# ホームページ ラフ案（ワイヤーフレーム）

## 1. ヘッダー（固定ナビゲーション）
```
┌──────────────────────────────────────────────────────────┐
│ [ロゴ]                    [HOME] [特徴] [メニュー] [アクセス] [お問い合わせ] │
└──────────────────────────────────────────────────────────┘
```
**含まれる要素:**
- サイトロゴ（左側）
- ナビゲーションリンク（右側）
  - 各セクションへのスムーススクロール
- スクロール時に背景色が変わるアニメーション


## 2. メインビジュアル（ヒーローセクション）
```
┌──────────────────────────────────────────────────────────┐
│                                                          │

│                                                          │
│                    メインキャッチコピー                       │
│                  「〇〇〇〇〇〇〇〇〇〇」                      │

│                                                          │
│                   [CTAボタン：詳しく見る]                    │
```

---

## 推奨する動きのある機能:
1. **スムーススクロール** - ナビゲーションクリックで滑らかにセクション移動
2. **パララックス効果** - 背景画像が異なる速度でスクロール
3. **フェードインアニメーション** - スクロールで要素が順次表示
4. **ホバーエフェクト** - ボタンやカードにマウスを乗せると動く
5. **ハンバーガーメニュー** - モバイル対応のメニュー
6. **ローディングアニメーション** - ページ読み込み時の演出
7. **画像ギャラリー** - クリックで拡大表示（ライトボックス）
