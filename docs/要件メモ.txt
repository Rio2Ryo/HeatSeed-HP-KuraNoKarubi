# 「蔵のカルビ 寒川店」HP制作ミーティングまとめ（確定版）

## 1. 概要
「蔵のカルビ 寒川店」は、  
家族連れや接待・会食をターゲットにした、**味の良さとコスパの高さが魅力の焼肉店**。  
上質な料理を落ち着いた空間で楽しめるよう、黒と金を基調に高級感を演出する。

---

## 2. コンセプト・方向性
- テーマ: 「高級感 × 家族の温かみ」  
- キャッチコピー候補: 「美味しいを、子どもにも。」  
- デザイン印象: 黒を基調に金色で上質さを表現。  
- 子どもへの配慮: 高級感がありながら、家族連れでも安心して過ごせる雰囲気。  
- 写真方針: 子どもと一緒に楽しむ焼肉シーンや、家族の団欒を感じる写真を掲載（撮影OK）。  
- 動画・画像: すでに受領済み（トップ用動画・店舗写真あり）。

---

## 3. ターゲット層
- 家族連れ（子ども連れ）  
- 接待・会食利用者  
※優先的に「家族連れ × 高級感」を重視。

---

## 4. 特徴・こだわり
- 味と価格のバランスが良く、コストパフォーマンスの高さが魅力。  
- 仕入れから提供まで、オーナーが一人ひとりに合わせてこだわる。  
- 個室が4室あり、プライベート空間でゆっくりと食事が可能。  
- 高級感のあるお酒も多数取り揃え。

---

## 5. デザイン・ビジュアル案
| 項目 | 内容 |
|------|------|
| カラーテーマ | 黒ベース × 金アクセント |
| フォント | 落ち着いた明朝体またはゴシック体 |
| トップ画像 | 高級感のある料理やお酒の写真、もしくは動画 |
| 動画 | トップページに短い紹介動画を掲載（受領済み） |
| 写真 | 家族でも楽しめる雰囲気と上品さを両立 |

---

## 6. 機能・コンテンツ案
| 項目 | 内容 |
|------|------|
| 予約機能 | 食べログ・ぐるなびの予約ページへのリンク誘導 |
| お客様の声 | Googleマップのレビューを引用掲載 |
| アクセス | GoogleマップからHPへリンク誘導、HPにも地図を埋め込み |
| メニュー紹介 | 高級感のある焼肉・お酒メニューを写真とともに掲載 |
| ストーリー | オーナーのこだわり・理念を文章で表現 |
